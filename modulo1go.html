Módulo 1 - Instalação e configuração

Passo 1. Instalar o Go

O primeiro passo é instalar o Go a partir da página oficial. Isso instala o compilador, a Biblioteca Padrão e muitas ferramentas para executar várias

tarefas comuns durante o desenvolvimento do Go.

No seu navegador, siga o link: go.dev/doc/install.

Baixe a versão para o seu sistema operacional. Uma vez baixado, execute o instalador, abra o prompt de comando e execute go version para

confirmar se o Go foi instalado.


Passo 2. Instalar o VS Code

Em seguida, instale o Visual Studio Code, que fornece funcionalidade básica de edição de código.

Abra seu navegador e pesquise por "vscode". Abra o primeiro link e baixe a versão para o seu sistema operacional. Uma vez baixado, execute o instalador.


Passo 3. Instalar e atualizar a extensão Go

O Visual Studio Code e a extensão Go fornecem IntelliSense, navegação de código e depuração avançada.

No Visual Studio Code, abra o modo de exibição Extensões clicando no ícone Extensões na Barra de Atividades. Ou use o atalho de teclado (Ctrl+Shift+X). Procure a extensão Go e selecione instalar.

Abra a aba Ajuda>da Paleta de Comandos Mostrar Todos os Comandos, ou use o atalho de teclado (Ctrl+Shift+P)

Procure por Go: Install/Update tools execute o comando a partir do palete. Quando solicitado, selecione todas as ferramentas Go disponíveis e selecione OK, e aguarde até que as ferramentas Go terminem de atualizar.
